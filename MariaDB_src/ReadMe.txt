Connecting to a MariaDB database from Lazarus/Free Pascal applications:

  - MariaDB1: Using a TMySQL57Connection; needs libmysql.dll (MySQL 5.7 client), shiped with MySQL (copy available at www.streetinfo.lu),
              to be copied to the project output directory.
  - MariaDB2: Using a TODBCConnection; driver available for download at the MariaDB website.

For detailed description concerning these two projects, please visit: http://www.streetinfo.lu/computing/lazarus/project/use_mariadb.html.
