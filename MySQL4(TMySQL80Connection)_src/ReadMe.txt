Writing data to a MySQL database from Lazarus / Free Pascal applications:

  - MySQL4: This sample application, intended for MySQL 8.0, uses the TMySQL80Connection object (available with the actual versions of Lazarus) to connect to MySQL;
            the required DLL libmysql.dll is shiped with the MySQL installation files and has to be copied to the project output directory.

For a detailed description concerning this project, please visit: http://www.streetinfo.lu/computing/lazarus/project/write_mysql.html.
