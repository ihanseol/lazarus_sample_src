This archive contains the application samples used with the tutorial "Using Microsoft Access databases with Lazarus/Free Pascal" at
  https://www.streetinfo.lu/computing/lazarus/programming/msaccess.html

Please, note that these applications have been developed and tested using MS Access inluded with Microsoft Office XP Pro
on a Windows XP Pro (32-bit) computer. I think, however, that the source code should also apply to the actual version of
Access on Windows 10/11...
