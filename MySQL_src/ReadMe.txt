Connecting to a MySQL database from Lazarus / Free Pascal applications:

  - MySQL1: Using a TMySQLxyConnection (needs libmysql.dll, shiped with MySQL, to be copied to project output directory).
  - MySQL2: Using a TODBCConnection (MS Windows: drver is included in "MySQL Installer" package).

For detailed description concerning these two projects, please visit: http://www.streetinfo.lu/computing/lazarus/project/connect_mysql.html.
