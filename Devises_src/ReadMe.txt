Mise-à-jour des cours de change.
================================

Les cours de change sont mis à jour par téléchargement d'un fichier du site de la Banque de France (www.banque-france.fr).
Ce téléchargement est réalisé en appelant le programme console wget, NON PAS inclus avec le code source de l'application. Vous pouvez le
trouver sur le site de Free Software Foundation, Inc. ou tout simplement télécharger le fichier .zip contenant l'exécutable de l'application
"Devises" de mon site (http://www.streetinfo.lu/computing/lazarus/files/Devises.zip); le fichier inclut wget.exe + documentation.

--------------------------------------------------------------------------------------------------------------------------------------------

Currency rates update.
======================

Currency rates are updated by downloading the previous workday rates from the site of "Banque de France" (www.banque-france.fr).
This download is done by calling the command line program wget, NOT included with the source code of the application. You can find it on the
site of Free Software Foundation, Inc. or simply download the .zip archive, containing the executable of the application "Devises" from my
website (http://www.streetinfo.lu/computing/lazarus/files/Devises.zip); the archive contains wget.exe + documentation.
