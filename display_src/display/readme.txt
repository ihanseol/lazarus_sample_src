This ZIP contains the sample source files used with the tutorial "Building and using Free Pascal dynamic link libraries" at streetinfo.lu.
The DLL display.dll contains some display functions based on the Crt unit. The three programs test_display, test_display2, and test_display3
show how to call these procedures from Pascal, C, resp. C++.
URL of the tutorial: https://www.streetinfo.lu/computing/lazarus/project/fpc_dll.html
