This archive contains simple Lazarus/Free Pascal projects to connect to MySQL resp. MariaDB using Zeos components. In fact, it's the
source code of the application samples of the tutorial "Connecting from Lazarus/Free Pascal to MySQL/MariaDB using Zeos components" at
  https://www.streetinfo.lu/computing/lazarus/project/connect_zeos.html

To use the MySQL application, you'll have to copy the file libmysql.dll to your project directory.
To use the MariaDB application, you'll have to copy the file libmariadb.dll to your project directory.
  -> Please, visit the tutorial page for details.
